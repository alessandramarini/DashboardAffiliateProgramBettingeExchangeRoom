Enterprise Cybersecurity

Threat modeling and automated defense mechanisms.

1. On-Chain Signatures

claim() MUST verify ED25519 signature from backend: Hash(Amount, Wallet, Nonce).

2. Strict Deduplication

DB enforces UNIQUE constraint on TON tx_hash.

3. Wallet Change Time-Lock

Updating Active_Wallet triggers a 72-hour freeze on withdrawals.

4. Admin Thresholds

Payouts > 1,000 USDT get flagged as Pending_Manual_Review.

5. Offline Cold Storage

Smart contract has an emergency sweep function to move funds to a hardware wallet.

Anti-Fraud Math Engine

The primary attack vector is Payload Spoofing: a malicious user alters the frontend to send an 'Elite' payload (180 days) but only attaches 'Starter' funds (47 USDT). The backend ignores the text payload and adjusts days based solely on the real on-chain collateral.
